# ☕ Java desde Cero

Repositorio personal para aprender **Java desde cero**, con enfoque práctico y progresivo.  
Aquí documento mi proceso de aprendizaje con ejercicios, ejemplos y mini proyectos.

## 🎯 Objetivo
Construir bases sólidas en Java a través de la práctica constante, resolviendo problemas reales y entendiendo los fundamentos del lenguaje.

## 🧭 Contenido
- Fundamentos de Java (variables, tipos de datos, operadores)
- Control de flujo (if, switch, bucles)
- Métodos y clases
- Programación orientada a objetos (POO)
- Estructuras de datos básicas
- Ejercicios prácticos
- Mini proyectos
